This repository contains Stata code to replicate the analyses in "Gender egalitarianism and marital dissolution: understanding the bifurcation of the 'gender revolution' in the United States."
The user needs to download the corresponding PSID data themselves - which is freely and publicly available once the user has created an account - from https://psidonline.isr.umich.edu/
To run the code, the macros in the setup file need to be changed to direct to the corresponding locations on your personal computer.
